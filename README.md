# Voting Eligibility Checker

This is a simple Java program that checks whether a person is eligible to vote.

## Concepts Used
- Variables
- Scanner Class
- User Input
- If-Else
- Relational Operators

## How It Works
- User enters age
- Program checks if age is 18 or above
- Displays eligibility result

## Output Example
Enter your age: 20
Eligible for voting
